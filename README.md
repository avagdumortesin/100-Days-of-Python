# 100 Days of Python
This repo is a collection of the daily projects from the AppBrewery.io course 100 Days of Code™: The Complete Python Pro Bootcamp on Udemy.

Within each folder, you will find a `main.py` file that contains the code for that day's project. In some of the earlier days, there may be code that was written by the AppBrewery staff, but the majority of it is what I wrote. In any cases where there are additional files, there will be a `README.md` file if I wrote them; if there is not a README, the course provided those files.
